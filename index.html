Dashboard de Cumplimiento: Demanda vs Venta vs Abastecimiento

Objetivo: Desarrollar una herramienta visual que permita al área de Supply Chain comparar la demanda proyectada, las ventas reales y las órdenes de compra, facilitando la toma de decisiones para el abastecimiento oportuno.

Herramientas utilizadas: QlikView · SQL Server · Power Automate

Rol: Líder de proyecto y desarrollador principal

Descripción: Diseñé e implementé un dashboard interactivo que compara las piezas proyectadas (FCST) con las piezas ordenadas (PO) a lo largo de las semanas del año. El tablero incluye filtros dinámicos por clase, cliente, región y tendencias, y permite a los usuarios explorar los datos de forma detallada. También se integró con Power Automate para automatizar actualizaciones y alertas.

Impacto: El área de Supply Chain utiliza activamente este dashboard para monitorear el cumplimiento de la demanda y ajustar el abastecimiento, mejorando la eficiencia operativa y reduciendo riesgos de desabasto.

RPA Logística Aérea de Flores

Objetivo: Automatizar la extracción, limpieza y consolidación de datos logísticos provenientes de múltiples fuentes para generar reportes visuales y alertas que optimicen la planificación de itinerarios aéreos.

Herramientas utilizadas: Power Automate · Python (FastAPI, Playwright) · API REST · Excel · Power BI · Alertas en Microsoft Teams

Rol: Líder de proyecto y desarrollador principal

Descripción: Diseñé e implementé una solución RPA que automatiza la recopilación de datos logísticos desde portales web, archivos Excel y APIs. Los datos son procesados mediante FastAPI y Playwright, y luego consolidados en reportes visuales en Power BI. Se integraron alertas automáticas vía Teams para notificar sobre cambios críticos en itinerarios o costos.

Impacto: La solución redujo significativamente la carga operativa manual, mejoró la precisión en la planificación logística y permitió una visualización clara de costos por aerolínea, rutas y semanas.
